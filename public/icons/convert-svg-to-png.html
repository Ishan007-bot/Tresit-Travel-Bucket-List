Instructions:

  1. The SVG icon is displayed on the left, and it will be converted to PNG on the right.
  2. Click the "Download PNG" button to download the icon as a PNG file.
  3. Place this PNG file in the icons directory to use it in your PWA.
  4. Repeat this process for each icon size in your manifest.json file.

Recommended icon sizes:

  • 72x72
  • 96x96
  • 128x128
  • 144x144

  • 192x192
  • 384x384
